# minerals

These are not rocks THESE ARE MINERALS, MARIE!

This repo is a structured mineral knowledge base built for weird creative software, generative art systems, visual prompt engines, educational tools, and mineral-obsessed apps.

It is meant to hold both:

1. **Scientific mineral data**
   - formula
   - crystal system
   - atomic / structural notes
   - hardness
   - luster
   - cleavage / fracture
   - formation environment
   - associated minerals
   - localities
   - variety relationships

2. **Aesthetic / symbolic / app-facing data**
   - color variants and what causes them
   - patterns like banded, botryoidal, druzy, fibrous, etc.
   - vibe / mood tags
   - palette tags
   - cultural associations
   - birthstone / zodiac / historical uses
   - search tags
   - prompt fragments
   - visual identity notes

---

## Repo Philosophy

This is not just a geology repo.

It is also:
- a visual texture library
- a symbolic archive
- a chemistry-to-aesthetic translation machine
- a metadata pile for creative apps
- a way to make minerals usable as structured visual ingredients

---

## Main Principles

- One mineral species per file
- Varieties can get their own files when useful
- Keep scientific facts separate from interpretive tags
- Use sources for factual claims
- Use controlled vocabularies where possible
- Optimize for both humans and apps
